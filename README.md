A streamlined web application designed to estimate pension benefits based on key user inputs such as entry age, gender, retirement age, starting salary, and annual salary growth. This tool helps users visualize pension projections using fundamental actuarial principles. It supports two pension program options: individual pension and pension with a partner.

With the Pension Calculator, you can obtain:

- Estimated monthly pension benefit results
- A salary growth projection chart throughout the working period

This calculator runs entirely in the browser, requires no external dependencies, and is deployed via GitHub Pages for fast and effortless access.
